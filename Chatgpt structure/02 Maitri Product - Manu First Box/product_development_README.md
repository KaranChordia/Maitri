# Maitri Product Development

This folder holds the physical product and revenue architecture for Maitri.

Use this folder for:

- First doll box design.
- Storybook packaging.
- In-box activities and stickers.
- Add-on accessory planning.
- Manufacturing readiness.
- Product testing.
- Cost, pricing, and margin assumptions.
- Safety, material, and compliance notes.

## Current Product Principle

Start with one complete box experience before launching the full ecosystem.

The first product should feel rich enough for a child and parent, while keeping manufacturing complexity controlled.

## Current Priority

Manu is the first box anchor.

The launch box should include:

- Doll.
- 32-page storybook.
- Letter from Manu.
- Six activity pages.
- Sticker pages.

Everything else should be mapped as a separate add-on or later expansion.
