# Project Status

## Current Phase

Foundation retainer implementation.

Latest strategic clarification:

Maitri should not scale the full ecosystem on day one. The next foundation layer is one complete Manu box experience: doll, 32-page storybook, letter, activities, and stickers. Accessories, playsets, games, workshops, and later books should be treated as sequenced add-ons after the first character experience is validated.

## Immediate Priorities

1. Run the 90-day foundation retainer around website validation, Manu's character universe, and Instagram + YouTube launch.
2. Use the website waitlist + parent survey to identify real interest before heavier product investment.
3. Build from Manu's completed character bible v1 and first story content set.
4. Prepare and review the month-one Instagram + YouTube content calendar.
5. Report monthly on audience segment mix, story preference, beta-reader interest, school interest, preorder signal, and objections.
6. Convert the new first-box approach into Manu's storybook blueprint, product box decisions, and validation tests before expanding into accessory production.
