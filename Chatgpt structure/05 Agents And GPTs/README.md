# Agents And GPTs

Use `agent_blueprints.md` to create the first four Maitri agents or GPTs:

1. Maitri Chief Of Staff.
2. Manu Story And Product Editor.
3. Maitri Content Producer.
4. Maitri Signal Analyst.

If workspace agents are available in ChatGPT Business, use them for shared or recurring workflows. If not, create these as GPTs first.

Do not create manufacturing, legal, school-sales, or second-character agents yet. Maitri is still in foundation mode.

## Copy-Ready Agent Folders

Each numbered folder contains:

- `_AGENT_INSTRUCTIONS_TO_COPY.md` for the agent setup screen.
- The files to upload as that agent's knowledge base.
- A best first test prompt.

Use these folders:

- `01 Maitri Chief Of Staff`
- `02 Manu Story And Product Editor`
- `03 Maitri Content Producer`
- `04 Maitri Signal Analyst`
