# ChatGPT Structure For Maitri

Use this folder to set up Maitri inside ChatGPT Business.

Each numbered folder corresponds to one ChatGPT Project or setup area.

## How To Use

1. Open ChatGPT Business.
2. Create each Project using the folder names below.
3. Upload the files from that matching folder into the Project.
4. Open `_PROJECT_INSTRUCTIONS_TO_COPY.md` inside the folder and paste that text into the Project instructions.
5. Use `05 Agents And GPTs/agent_blueprints.md` to create the first Maitri agents or GPTs.

## Folders

### 00 Setup Guide

Read this first. It explains the whole ChatGPT Business setup.

### 01 Maitri HQ - Strategy And Decisions

Create this as the main strategy and founder-alignment Project.

### 02 Maitri Product - Manu First Box

Create this as the product-development Project for Manu's first doll box and 32-page storybook.

### 03 Maitri Content Studio

Create this as the content Project for Instagram, YouTube Shorts, newsletters, and parent-facing storytelling.

### 04 Maitri Validation And Signals

Create this as the evidence Project for waitlist, survey, beta-reader, objection, and preorder-readiness analysis.

### 05 Agents And GPTs

Use this to create Maitri Chief Of Staff, Manu Story And Product Editor, Maitri Content Producer, and Maitri Signal Analyst.

## Important Rule

Do not upload every Maitri file into every ChatGPT Project.

Each Project should stay focused so Shwethika can understand and use it easily.
