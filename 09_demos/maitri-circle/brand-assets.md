# Maitri Brand Assets

Working brand notes for the investor-demo website in `09_demos/maitri-circle`.

## Color System

### Pink Family

Use pink as the emotional brand color: warmth, affection, courage, and the Manu companion feeling.

| Token | Hex | Usage |
| --- | --- | --- |
| Maitri Rose | `#ef6f8b` | Primary brand accent, icons, labels, nav highlights |
| Maitri Deep Rose | `#db5474` | Stronger accent, footer/details, pressed states |
| Button Rose | `#f46f8c` | Primary CTA gradient start |
| Button Deep Rose | `#ea617f` | Primary CTA gradient end |
| Layer Pink | `#ffd0df` | Hover backing layer behind homepage images |
| Soft Blush | `#fff1e7` | Warm section background and soft panels |
| Warm Pink Wash | `#ffe8ef` | Rose-tinted cards and atelier panels |

### Yellow Family

Use yellow as the optimism and keepsake color: launch energy, story warmth, small reward moments, and highlights.

| Token | Hex | Usage |
| --- | --- | --- |
| Maitri Amber | `#f3a823` | Secondary brand accent, focus ring, label accents |
| Soft Story Yellow | `#fff0dc` | Hover layer blend with pink |
| Cream | `#fff8ed` | Warm page sections, nav glass tint, soft CTA surfaces |
| Warm Cream | `#fff6ec` | Homepage gradient band |
| Soft Gold | `#fff6a6` | Tiny sparkle details on buttons |
| Pale Keepsake Yellow | `#fff3d5` | Accessory/keepsake card backgrounds |

## Recommended Pairing

The investor-demo site should lead with the pink family and use yellow as a supporting warmth/highlight system. Avoid making the page mostly yellow; it should appear as light, optimism, and shimmer around the rose-led brand.

## Typography

Current selected family:

| Role | Font | Reason |
| --- | --- | --- |
| Whole website | `Outfit` | Rounded, modern, friendly, and clean enough for a premium consumer product site |

Outfit should be used across brand, headings, body, nav, buttons, forms, and Story Universe UI so the website feels like one cohesive consumer brand system.
