# Maitri Website Demo

A polished pre-launch web app for Maitri Dolls.

## Purpose

The site introduces Maitri as a story-first companion doll universe, introduces Manu, invites families into the Maitri Circle, recruits beta readers and schools, and captures early waitlist demand before inventory investment.

## Run Locally

```bash
node server.js
```

Then open:

```text
http://localhost:4173
```

## Structure

- `index.html` - page structure and semantic content.
- `src/styles.css` - full visual system, layout, responsive design, and motion.
- `src/app.js` - interactions, local waitlist state, smooth scrolling, and reveal behavior.
- `assets/` - project-bound website assets.
- `design_references/` - generated concept reference used for implementation.
